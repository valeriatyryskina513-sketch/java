Лабораторная работа №4: Исключения в Java
Описание

Лабораторная работа посвящена обработке исключений в Java.

Изучены:

-try-catch-finally;

-стандартные исключения;

-создание собственных исключений.

Структура проекта

1. ArrayAverage.java — Среднее арифметическое массива

Программа
считывает массив,
вычисляет среднее значение,
обрабатывает ошибки ввода.

Используются исключения:

-NumberFormatException;

-ArrayIndexOutOfBoundsException.

2. FileCopy.java — Копирование файлов

Программа 
копирует содержимое файла,
работает с FileReader и FileWriter,
обрабатывает ошибки работы с файлами.

3. CustomEmptyStackException.java — Пользовательское исключение

Создано собственное исключение для обработки пустого стека.

4. CustomStack.java — Реализация стека

Реализованы методы
push(),
pop().

Также реализовано логирование исключений в файл.

5. Main.java — Проверка работы стека

Демонстрируется:

-выброс собственного исключения;

-обработка ошибки;
-запись информации в файл.

Результат работы

Были изучены механизм исключений Java, создание собственных классов ошибок, обработка ошибок файлов и массивов.
