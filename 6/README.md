Лабораторная работа №6: Коллекции и дженерики в Java
Описание

Лабораторная работа посвящена изучению коллекций Java, дженериков и итераторов.

В ходе работы были реализованы программы для:

-анализа текста;

-работы со стеком;

-учёта продаж.

Структура проекта

1. TopWords.java — Анализ текста

Программа считывает текстовый файл и определяет топ-10 самых популярных слов.

Используются HashMap и Collections.sort()

2. Stack.java — Обобщённый стек

Реализован класс: Stack<T>

Методы: push(), pop(), peek()

3. StackTest.java — Проверка работы стека

Проверяется:

-добавление элементов;

-извлечение элементов;

-работа дженериков.

4. Product.java — Класс товара

Описывает название товара и цену товара.

5. SalesManager.java — Учёт продаж

Реализованы:

-хранение продаж;

-вычисление общей суммы;

-поиск популярного товара.

6. Main.java — Проверка программы

Выполняется тестирование системы продаж.

Результат работы

Были изучены коллекции List, Map, дженерики, сортировка данных, работа с объектами коллекций.
