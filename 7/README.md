Лабораторная работа №7: Многопоточность в Java
Описание

Лабораторная работа посвящена изучению многопоточности и синхронизации потоков.

Изучены:

-создание потоков;

-методы start() и join();

-синхронизация;

-работа с общими ресурсами.

Структура проекта

1. ArraySumThread.java — Сумма массива

Каждый поток вычисляет сумму части массива.

2. ArraySumMain.java — Работа потоков

Выполняется запуск потоков и объединение результатов.

3. RowMaxThread.java — Поиск максимума строки

Каждый поток обрабатывает отдельную строку матрицы.

4. MatrixMaxMain.java — Поиск максимума матрицы

Главный поток получает результаты потоков и определяет максимальный элемент.

5. Product.java — Описание товара

Содержит:

-название;

-вес товара.

6. Warehouse.java — Работа склада

Реализованы:

-очередь товаров;

-синхронизация доступа;

-ограничение веса.

7. Loader.java — Поток-грузчик

Каждый поток переносит товары и взаимодействует со складом.

8. WarehouseMain.java — Проверка программы

Создаются:

-склад;

-товары;

-потоки-грузчики.

Результат работы

Были изучены многопоточность, синхронизация потоков, работа с общими ресурсами, взаимодействие потоков.
