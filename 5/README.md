Лабораторная работа №5: Регулярные выражения в Java
Описание

Лабораторная работа посвящена использованию регулярных выражений в Java.

Изучены:

-классы Pattern и Matcher;

-поиск и обработка текста;

-проверка строк по шаблону.

Структура проекта

1. NumberFinder.java — Поиск чисел в тексте

Программа находит целые числа, дробные числа.

2. PasswordValidator.java — Проверка пароля

Проверяются:

-длина пароля;

-наличие цифр;

-наличие заглавных букв.

3. LowerUpperFinder.java — Поиск сочетаний букв

Находит случаи, когда после строчной буквы идёт заглавная.

4. IPValidator.java — Проверка IP-адреса

Проверяется корректность IPv4-адреса, диапазон значений от 0 до 255.

5. WordsStartingWithLetter.java — Поиск слов

Программа ищет слова на заданную букву.

Результат работы

Были изучены регулярные выражения, поиск совпадений, фильтрация текста, обработка строк в Java.
