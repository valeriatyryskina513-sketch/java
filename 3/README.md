Лабораторная работа №3: Хэш-таблицы и HashMap

Описание

Лабораторная работа посвящена изучению принципов работы хэш-таблиц и методов разрешения коллизий.

Были реализованы:

-собственная хэш-таблица;

-работа со встроенным классом HashMap.

Структура проекта

1. HashTable.java — Собственная реализация хэш-таблицы

Реализованы методы:
put(),
get(),
remove(),
size(),
isEmpty().

Используется:

-метод цепочек;

-LinkedList;

-обобщённые типы <K, V>.

2. Student.java — Класс студента

Содержит:
имя,
фамилию,
возраст,
средний балл.

3. StudentFleet.java — Работа с HashMap

Используется коллекция:
HashMap<String, Student>

Реализованы методы:
добавления,
поиска,
удаления студентов.

4. Main.java — Проверка работы программы

Выполняется тестирование всех операций хэш-таблицы.

Результат работы

Были изучены
принципы хэширования,
коллизии,
структуры данных «ключ–значение»,
коллекции Java.
