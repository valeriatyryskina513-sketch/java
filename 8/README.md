Лабораторная работа №8: Аннотации, Stream API и java.util.concurrent
Описание

Лабораторная работа посвящена:

-аннотациям;

-Stream API;

-многопоточной обработке данных;

-механизму рефлексии.

Структура проекта

1. DataProcessor.java — Пользовательская аннотация

Создана аннотация @DataProcessor

Используется: @Retention(RetentionPolicy.RUNTIME)

2. FilterProcessor.java — Фильтрация данных

Метод удаляет короткие строки.

Используется:

-stream()

-filter()

3. TransformProcessor.java — Преобразование строк

Метод переводит строки в верхний регистр.

Используется: map(), toUpperCase()

4. AggregateProcessor.java — Добавление префикса

К каждой строке добавляется: "Обработано:"

5. DataManager.java — Менеджер обработки данных

Реализованы методы:

-registerDataProcessor()

-loadData()

-processData()

-saveData()

-printData()

Используются:

-ExecutorService

-Future

-рефлексия;

-многопоточность.

6. Main.java — Главный класс

Выполняется регистрация обработчиков, загрузка файла, запуск обработки, сохранение результата.

Результат работы

Были изучены аннотации, Stream API, пул потоков, механизм рефлексии, многопоточная обработка данных.
