Лабораторная работа №2: Объектно-ориентированное программирование в Java
Описание

Лабораторная работа посвящена изучению основных принципов объектно-ориентированного программирования:

абстракции,
инкапсуляции,
наследования,
полиморфизма.

В качестве предметной области реализована иерархия классов велосипедов.

Структура проекта
1. Bicycle.java — Базовый абстрактный класс

Содержит:

-поля brand, model, color;

-статический счётчик объектов;

-конструкторы;

-геттеры и сеттеры;

-метод displayInfo();

-абстрактный метод ride().

2. MountainBike.java — Горный велосипед

Наследуется от Bicycle.

Дополнительно реализованы:

тип подвески,
тип тормозов,
тип покрышек,
переопределение методов.

3. KidsBike.java — Детский велосипед

Добавлены:

-дополнительные колёса;

-корзинка;

-рекомендуемый возраст;

-оформление велосипеда.

4. BMXBike.java — BMX велосипед

Реализованы:

материал рамы,
размер рамы,
наличие пег,
тип руля.

5. BicycleDemo.java — Проверка работы программы

Выполняется:

-создание объектов;

-демонстрация полиморфизма;

-вызов методов;

-работа со счётчиком объектов.

Результат работы

В ходе работы были успешно реализованы
наследование классов,
переопределение методов,
использование абстрактных классов,
работа со статическими полями и методами.
