Лабораторная работа №1: Знакомство с Java и базовые алгоритмы

Описание:

Данная лабораторная работа посвящена изучению базового синтаксиса языка Java, работе с методами, циклами и обработкой строк. В ходе работы были реализованы программы для поиска простых чисел и проверки строк на палиндром.
Структура проекта:
В проекте представлены две основные программы:

1. Primes.java — Поиск простых чисел
Программа выводит все простые числа в диапазоне от 2 до 100.
Реализован метод isPrime(int n) для проверки числа на простоту.

Используются:
цикл for
оператор остатка %
условные конструкции if

2. Palindrome.java — Проверка палиндромов
Программа анализирует строки на наличие свойства палиндрома.

Реализованы методы:
reverseString(String word) — переворачивает строку;
isPalindrome(String word) — сравнивает исходную строку и перевёрнутую.

Для обработки строк используются:
charAt()
length()
equals()
Результат работы

Программа корректно:
определяет простые числа;
выполняет реверс строки;
определяет палиндромы среди введённых слов.
